# DFIR Projects

This section contains digital forensics and incident response (DFIR) case studies derived from academic and independent investigations, designed to demonstrate practical forensic methodology, analytical reasoning, and professional reporting.

Focus areas include:
- Disk and USB forensic investigations
- Live response, memory, and process analysis
- Timeline reconstruction and artifact correlation
- Evidence handling and integrity validation
